# 登陆状态

## HTTP

1. HTTP 1.0 是无状态的， 每一次请求相应都是独立的

### 实现连接方式

1. 要保持连接状态，需要一个标识来实现 (ip, uuid, agent, token)
2. HTTP 1.0 实现的连接，轮询(不停的请求)
    1. 客户端 问 服务端： 有没有信息啊
    2. 服务端一直返回： 还没有，再等等
3. HTTP 1.1 协议中的长连接 keep-alive 头信息, 能够建立浏览器和服务端的一个长连接
4. web socket 协议
